# Installation of PySide2
PyPI를 통해서 코드 한 줄로 인스톨 하는 것이 가능하다.
가상환경을 구축하는 것을 추천한다.

## Windows & MacOS
```
pip3 install pyside2
```

## Linux(Ubuntu)
```
apt-get install python3-pyside2
```
